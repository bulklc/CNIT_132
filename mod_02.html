Module 2 - Design, SEO, WebDAV

This web page has been created in response to instructor direction provided on the Module 2 Assignment Page on Canvas. This assignment is due by 11:59 pm on February 9th, 2021. The total amount of non-extra-credit points achievable is 10.

Search Engine Optimization (SEO) is an aspect of website development whose importance cannot be overemphasized. According to the Webopedia glossary entry on the subject,
"...SEO is the process of improving a website or web page so it increases organic traffic quality and quantity from search engines."
This entry goes on to explain that a website or webpage developed with SEO as a priority will have improved chances of appearing on any given search engine's result page (SERP), even when the result is not "pushed" to the top of the results as a "promoted" ad.

Finally, this entry details several factors which play into a site's SEO. Some, but certainly not all, are detailed below:
  • Keyword Density

    A page with keywords which repeat often is more likely to be recognized as a relevant result for any given search engine which seeks out such terms. It should be noted, however, that many search engines are "smart" enough to recognize when keywords are not used properly (consecutive repeats, out-of-context placement, etc). Thus, only appropriately crafted key terms will yield an optimize search engine interaction.
  • Metadata

    A page with detailed metadata, which summarizes page content, is recognized by many search engines as a higher-quality, more relevant search result. This includes all lines within the <head> tag which start with the <meta> tag.

    This includes:
    • <meta charset="UTF-8">
    • <meta name="description" content="Page Description">
    • <meta name="keywords" content="Page Key Word A, Page Key Word B">
    • <meta name="author" content="Page Author">
    • <meta name="viewport" content="width=device-width, initial-scale=1.0">
  • Links

    Pages with generous internal links, external links, and back links are considered to have more credibility as they strategically route viewers to proper online sources.
  • Security

    Pages which load with a proper SSL certificate are considered safer and more secure, thus a better result to return to a user on a given search engine.
  • Speed

    Search engines consider pages which load quickly to be favorable results for their users. Therefore, a pages load-time should be minimized in pursuit of SEO. This can be achieved by reducing content size, or having some elements load only when the user requests them to (via interactive elements such as buttons, etc).
